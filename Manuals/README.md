## Σημείωση

Τα εγχειρίδια αφορούν **μόνο** τη λειτουργικότητα που είναι διαθέσιμη στο staging. Η αυτοεγγραφή χρηστών και η σύνδεση μέσω SSO **δεν** καλύπτονται — θα προστεθούν σε μεταγενέστερη έκδοση.
